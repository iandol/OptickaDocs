Detailed Description

optickaCore base class inherited by many other opticka classes.

Introduction

optickaCore is itself derived from handle. It provides methods to find attributes with specific parameters (used in autogenerating UI panels), clone the object, parse arguments safely on construction and add default properties such as datestamp, UUID and name/comment management.
