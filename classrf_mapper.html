rfMapper Class Reference

receptive field mapper rfMapper is a mouse driven receptive field mapper, using various keyboard commands to change the stimulus and and ability to record where the mouse is (i.e. draw) in screen co-ordinates to save as a hand map image for storage. It is based on barStimulus, and you can change length + width (h, j, l & k on keyboard) and texture of the bar (space), and angle (arrow keys), and colour of the bar and background (numeric keys). Right/middle mouse draws the current position into a buffer, you can turn on and off visualising drawing buffer with ; and use ' to reset the drawn positions. See the checkKeys method for better description of what the keyboard commands do. More...

Detailed Description

receptive field mapper rfMapper is a mouse driven receptive field mapper, using various keyboard commands to change the stimulus and and ability to record where the mouse is (i.e. draw) in screen co-ordinates to save as a hand map image for storage. It is based on barStimulus, and you can change length + width (h, j, l & k on keyboard) and texture of the bar (space), and angle (arrow keys), and colour of the bar and background (numeric keys). Right/middle mouse draws the current position into a buffer, you can turn on and off visualising drawing buffer with ; and use ' to reset the drawn positions. See the checkKeys method for better description of what the keyboard commands do.
